SVWORLD Standard 73

**Проект:** HTML-сайт с методикой SVWORLD Standard 73 Rev.73.2  

**Описание:**
- Таблицы CVR₇₃ по странам (SRI₇₃, CVR₇₃, Zone)  
- Разделы про коммерческие и центральные банки  
- Информация о мировом долге (~348 трлн $USD) и методика снижения долгов  
- Поддержка 13 языков интерфейса: английский, русский, румынский, французский, китайский, хинди, японский, арабский, тайский, казахский, иврит, турецкий, испанский  

**Как открыть сайт на планшете или ПК:**
1. Перейти в папку с файлом `svworld.html`  
2. На iPad / iPhone:  
   - Долгий тап на файле → **Открыть в Safari**  
3. На ПК / Mac:  
   - Двойной клик на файле → откроется в браузере (Chrome, Edge, Firefox, Safari)
